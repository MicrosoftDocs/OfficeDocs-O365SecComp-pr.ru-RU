## <a name="mail-flow-rule-components"></a><span data-ttu-id="86709-122">Компоненты правила обработки почтового потока</span><span class="sxs-lookup"><span data-stu-id="86709-122">Mail flow rule components</span></span>

- <span data-ttu-id="86709-p103">**Условия** указывают, с какими сообщениями должны выполняться действия. Некоторые условия проверяют поля заголовков сообщений (например, поля "Кому", "От" или "Копия").

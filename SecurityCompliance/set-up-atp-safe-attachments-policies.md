# <a name="set-up-office-365-atp-safe-attachments-policies"></a>Настройка политик Office 365 ATP безопасные вложения

Пользователи регулярно отправлять, получать и обмен информацией вложения, таких как документы, презентации, электронные таблицы и другие. Не всегда легко определить, является ли вложение безопасных или намеренного по сообщения электронной почты. И самое последнее, который будет вредоносных вложений для получения, замедляет для вашей организации. К счастью могут помочь [Защиту от угроз для Office 365 Advanced](office-365-atp.md) (ATP). Можно настроить [Безопасные вложения ATP](atp-safe-attachments.md) политик чтобы обеспечить защиту вашей организации от атак с вложениями небезопасных электронной почты. 
  
## <a name="what-to-do"></a>Действия 
  
1. [Просмотрите необходимого программного обеспечения](#review-the-prerequisites)
    
2. [Настройка политики почтовых ATP безопасные вложения](#set-up-an-atp-safe-attachments-policy)
    
3. [Сведения о параметрах политики ATP безопасные вложения](#learn-about-atp-safe-attachments-policy-options)
    
## <a name="step-1-review-the-prerequisites"></a>Шаг 1: Просмотрите необходимого программного обеспечения

- Убедитесь в том, что в вашей организации есть [Защиту от угроз для Office 365 расширенный](office-365-atp.md).
    
- Убедитесь в том, что у вас есть необходимые [разрешений, назначенных безопасности Office 365 &amp; центре соответствия требованиям](permissions-in-the-security-and-compliance-center.md).
    
- [Дополнительные сведения о параметрах политики ATP безопасные вложения](#learn-about-atp-safe-attachments-policy-options) (в этой статье). Некоторые параметры, такие как параметры монитора или заменить, может привести незначительные задержку электронной почты во время вложения, выполняется поиск вирусов. Чтобы избежать задержек сообщение, рекомендуется использовать [динамические доставки и предварительный просмотр](dynamic-delivery-and-previewing.md).
    
- Разрешить до 30 минут для новых или обновленных политики для распространения для всех центров обработки данных Office 365.
    
## <a name="step-2-set-up-or-edit-an-atp-safe-attachments-policy"></a>Шаг 2: Настройка (или изменение) политику ATP безопасные вложения
  
1. Как глобальный администратор или администратор безопасности, перейдите к [https://protection.office.com](https://protection.office.com) и войдите с учетной записи рабочего или школы. 
    
2. В Office 365 безопасность &amp; центре соответствия требованиям в левой области навигации, в разделе **Управление угроз**, выберите **политику** \> **Безопасные вложения**.
    
3. Если **Включить ATP для SharePoint, OneDrive и группами Майкрософт**, рекомендуется выбирать этот параметр. Это позволит [Office 365 расширенного защиту от угроз для SharePoint, OneDrive и группами Майкрософт](atp-for-spo-odb-and-teams.md) для вашей среды Office 365. 
    
4. Нажмите кнопку **Создать** (новой кнопки имеет следующий вид: знак плюс ( **+**)) для создания политики.
    
5. Укажите имя, описание и параметров для политики.
    
    **Пример:** Чтобы настроить политику, называется «без задержки», который предоставляет все сообщения сразу же и затем снова подключает вложения после их в случае сканирования, можно указать следующие параметры: 
    
      - В поле **имя** введите без задержки.
    
      - В поле **Описание** введите описание как немедленно выполняет доставку сообщений и вложений повторно присоединяет после сканирования.
    
      - В разделе ответа выберите параметр **Динамического доставки** . ([Дополнительные сведения о динамических доставки и предварительный просмотр с ATP безопасные вложения](dynamic-delivery-and-previewing.md)).
    
      - В разделе **Перенаправление вложения** выберите параметр для включения перенаправления и введите адрес электронной почты Office 365 глобального администратора, администратора безопасности или аналитик безопасности анализ вредоносных вложений. 
    
      - В разделе **Применить к** выберите **домен получателя — это**и затем выберите ваш домен. Последовательно выберите пункты **Добавить**и затем нажмите **кнопку ОК**.
    
6. Нажмите **Сохранить**.
    
Попробуйте настроить несколько политик ATP безопасные вложения для вашей организации. Эти политики будут применяться в порядке, в котором они перечислены на странице **ATP безопасные вложения** . После определения или изменения политики Разрешить менее 30 минут для политик вступили в силу в центрах обработки данных Майкрософт. 
  
## <a name="step-3-learn-about-atp-safe-attachments-policy-options"></a>Шаг 3: Сведения о политике ATP безопасные вложения

При настройке политик ATP безопасные вложения, выбрать много параметров, включая монитор, блока, заменить, динамический доставки и т. д. В случае, если вам интересно об этих параметров выполните, в следующей таблице обобщаются каждого и эффект.
  
|**Вариант**|**Эффект**|**Используется, когда требуется:**|
|:-----|:-----|:-----|
|**Выкл.** <br/> |Не проверять вложения на наличие вредоносных программ  <br/> Не задержка доставки сообщений  <br/> |Отключить сканирование для внутренних отправителей, сканеры, факсы или смарт-узлов, которые будет отправлять известных, подходящие вложения  <br/> Избежать лишних задержек в маршрутизации почты внутренних  <br/> **Этот параметр не рекомендуется для большинства пользователей. Оно позволяет отключить безопасные вложения ATP сканирования для небольшой группы внутренних отправителей.**           |
|**Монитор** <br/> |Доставка сообщения с вложениями и затем отслеживает, что происходит с обнаруженных вредоносных программ  <br/> |Просмотреть, где переходит обнаруженных вредоносных программ в организации  <br/> |
|**Блок** <br/> |Не позволяет сообщения с вложениями обнаруженных вредоносных программ с переходом  <br/> Отправляет сообщения с обнаруженных вредоносных программ в [карантин в Office 365](manage-quarantined-messages-and-files.md) , где администратор безопасности или аналитик можно просмотрите и освобождать (или удаление) этих сообщений  <br/> Автоматически блокирует последующих сообщений и вложений  <br/> |Защищают организации повторяющиеся атак с помощью одного вложения вредоносных программ  <br/> |
|**Заменить** <br/> |Удаляет обнаруженных вредоносных программ вложения  <br/> Уведомляет получателей удаления вложения  <br/> Отправляет сообщения с обнаруженных вредоносных программ в [карантин в Office 365](manage-quarantined-messages-and-files.md) , где администратор безопасности или аналитик можно просмотрите и освобождать (или удаление) этих сообщений  <br/> |Вызывает видимости получателям, что вложения были удалены из-за обнаруженных вредоносных программ  <br/> |
|**Динамические доставки** <br/> |Доставка сообщения сразу же  <br/> Заменяет вложений с помощью файла заполнитель, пока проверка завершена и затем снова подключает вложений, если не обнаружена вредоносная  <br/> Включает в себя Предварительный Просмотр возможностей для большинства Office и PDF-файлов вложений файлов во время поиска вирусов  <br/> Отправляет сообщения с обнаруженных вредоносных программ на карантин, где администратор безопасности или аналитик можно просмотрите и освобождать (или удаление) этих сообщений  <br/> [Сведения о динамических доставки и предварительный просмотр с ATP безопасные вложения](dynamic-delivery-and-previewing.md) <br/> |Избежать задержек сообщения получателям защиты от вредоносных файлов  <br/> Включение получателей для предварительного просмотра вложений в безопасном режиме во время сканирования  <br/> |
|**Включение перенаправления** <br/> |Применяется, если выбрано значение монитор, блокирование и заменить  <br/> Отправляет вложения на указанный адрес электронной почты, где можно изучить администраторов безопасности или аналитики  <br/> |Разрешить администраторов безопасности и аналитикам исследование подозрительные вложения  <br/> |

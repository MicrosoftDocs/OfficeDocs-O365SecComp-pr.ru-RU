# <a name="enable-unlimited-archiving-in-office-365---admin-help"></a>Включение неограниченном архивировании в Office 365 — Admin справки

Можно использовать Exchange Online развертываемым архивации функцию в Office 365 для включения неограниченное дискового пространства для архивных почтовых ящиков. При включении развертываемым архивации дополнительного места хранения автоматически добавляется архивный почтовый ящик пользователя при приближении максимальный объем дискового пространства. Результатом является емкость хранилища почтовых ящиков не ограничен. Чтобы включить автоматическое развертывание архивации для всех пользователей в вашей организации или только для определенных пользователей. Дополнительные сведения о расширении автоматическое архивирование, просматривать [Общие сведения о неограниченном архивировании в Office 365](unlimited-archiving.md).

## <a name="before-you-begin"></a>Перед началом работы

- Необходимо быть глобальным администратором в организации Office 365 или быть членом группы ролей управления организацией в организации Exchange Online для включения развертываемым архивации для всей организации или для отдельных пользователей. Кроме того необходимо быть участником группы ролей, назначенной роли получателей почты для включения развертываемым архивации для определенных пользователей.
    
- Архивный почтовый ящик пользователя должна быть включена перед включением развертываемым архивации. Пользователь должен быть назначен лицензии Exchange Online (план 2) для включения в архивный почтовый ящик. Если пользователь назначена лицензия на Exchange Online план 1, необходимо назначить их отдельная лицензия архивации Exchange Online для включения их архивного почтового ящика. Просмотреть [Enable архивных почтовых ящиков в Office 365 безопасность &amp; центре соответствия требованиям](enable-archive-mailboxes.md).
    
- Можно также использовать PowerShell для включения архивные почтовые ящики. Пример команды PowerShell, которые можно использовать для включения архивные почтовые ящики для всех пользователей в вашей организации в разделе [Дополнительные сведения](#more-information) . 
    
- Автоматическое развертывание архивации также поддерживает общие почтовые ящики. Чтобы включить в архив для общего почтового ящика, лицензии Exchange Online (план 2) или лицензии Exchange Online план 1 с лицензией архивации Exchange Online является обязательным.
    
- Нельзя использовать Центр администрирования Exchange или безопасности &amp; центре соответствия требованиям, чтобы включить автоматическое развертывание архивации. Необходимо использовать Exchange Online PowerShell. Чтобы подключиться к организации Exchange Online с помощью удаленной оболочки PowerShell, обратитесь к разделу [подключение к Exchange Online PowerShell](https://go.microsoft.com/fwlink/p/?linkid=396554).
    
  
## <a name="enable-auto-expanding-archiving-for-your-entire-organization"></a>Включить развертываемым архивацию для всей организации

Можно включить развертываемым архивацию для всей организации. После включения его, развертываемым архивации будет включена для существующих почтовых ящиков пользователей, а также для новых почтовых ящиков пользователей, которые будут созданы. При создании новых почтовых ящиков пользователей, не забудьте включить главного архивный почтовый ящик пользователя, функция архивации развертываемым будет работать для нового почтового ящика пользователя.
  
1. [Подключение к PowerShell Exchange Online](https://go.microsoft.com/fwlink/p/?linkid=396554)
    
2. Выполните следующую команду в Exchange Online PowerShell, чтобы включить развертываемым архивацию для всей организации.

    ```
    Set-OrganizationConfig -AutoExpandingArchive
    ```
  
## <a name="enable-auto-expanding-archiving-for-specific-users"></a>Включение развертываемым архивации для определенных пользователей

Вместо Включение развертываемым архивирования для каждого пользователя в вашей организации, можно только включить его для определенных пользователей. Это может сделать так, как только некоторые пользователи могут иметь потребность в очень большие архивного хранилища.
  
При включении развертываемым архивации для определенного пользователя и почтовый ящик пользователя в на удержание или назначения политики хранения к Office 365, выполняются следующие изменения две конфигурации:
  
- Квота хранилища для основного архивного почтового ящика пользователя увеличивается на 10 ГБ (от 100 ГБ на 110 ГБ). Квота предупреждений для архива также увеличивается на 10 ГБ (от 90 ГБ на 100 ГБ).
    
- Квота хранилища для папки элементов для восстановления в основной почтовый ящик пользователя увеличивается на 10 ГБ (также из 100 ГБ на 110 ГБ). Квота предупреждений элементов для восстановления также увеличивается на 10 ГБ (от 90 ГБ на 100 ГБ). Эти изменения могут быть применены только в том случае, если почтовый ящик в на удержание или назначенные политики хранения к Office 365.
    
В этом дополнительного пространства добавляется во избежание проблем хранения данных, которые могут возникнуть перед архив развертываемым подготовить к работе. Обратите внимание, что дополнительное хранилище пространства *не является* добавлена при включении развертываемым архивации для всей организации, как описано в предыдущем разделе. 
  
1. [Подключение к PowerShell Exchange Online](https://go.microsoft.com/fwlink/p/?linkid=396554)
    
2. Выполните следующую команду в Exchange Online PowerShell, чтобы включить развертываемым архивацию для определенного пользователя. Как объяснялось ранее архивного почтового ящика пользователя (основной архив) должна быть включена перед включением развертываемым архивации для этого пользователя.
    
    ```
    Enable-Mailbox <user mailbox> -AutoExpandingArchive
    ```


> [!IMPORTANT]
> В гибридном развертывании Exchange команда **Enable-Mailbox AutoExpandingArchive** нельзя использовать для включения развертываемым архивации для конкретных пользователей, основной почтовый ящик которого находится на локальном и их архивного почтового ящика на основе облака. Чтобы включить развертываемым архивацию для облачных архивных почтовых ящиков в гибридном развертывании Exchange, необходимо выполните команду **Set-OrganizationConfig AutoExpandingArchive** в Exchange Online PowerShell, чтобы включить развертываемым архивацию для во всей организации. Если пользователь основной и, и архивных почтовых ящиков на основе облака, можно использовать команды **Enable-Mailbox AutoExpandingArchive** для включения развертываемым архивации для определенных пользователей. 
  
## <a name="verify-that-auto-expanding-archiving-is-enabled"></a>Убедитесь, что включен параметр развертываемым архивации

Чтобы убедиться, что развертываемым архивации включена для вашей организации, выполните следующую команду в Exchange Online PowerShell.

```
Get-OrganizationConfig | FL AutoExpandingArchiveEnabled
```

Значение `True` развертываемым архивации включена для организации. 
  
Чтобы проверить, развертываемым архивации включить для определенного пользователя, выполните следующую команду в Exchange Online PowerShell.
  
```
Get-Mailbox <user mailbox> | FL AutoExpandingArchiveEnabled
```
Значение `True` развертываемым архивации включена для пользователя. 
  
После включения развертываемым архивации, сохранить в виду следующее:
  
- Если вы выполняете команду **Set-OrganizationConfig AutoExpandingArchive** , чтобы включить автоматическое развертывание архивации для вашей организации, не требуется выполнить **Enable-Mailbox AutoExpandingArchive** для отдельных почтовых ящиков. Обратите внимание на то, **Set-OrganizationConfig** используется для включения развертываемым архивации для вашей организации не изменяет свойство *AutoExpandingArchiveEnabled* на почтовые ящики пользователей в том, что работа `True`.
    
- Аналогично значения для свойств почтового ящика *ArchiveQuota* и *ArchiveWarningQuota* остаются прежними при включении развертываемым архивации. На самом деле при включении развертываемым архивацию для почтового ящика пользователя, свойство *AutoExpandingArchiveEnabled* задано значение `True`, свойства *ArchiveQuota* и *ArchiveWarningQuota* , просто игнорируются. Ниже приведен пример из этих свойств почтового ящика после развертываемым архивации включен для почтового ящика пользователя.

## <a name="more-information"></a>Дополнительные сведения

- Можно также использовать PowerShell для включения архивные почтовые ящики. Например вы может выполните следующую команду в Exchange Online PowerShell для включения архивные почтовые ящики для всех пользователей, чьи архивный почтовый ящик уже не будет включен.

    ```
    Get-Mailbox -Filter {ArchiveStatus -Eq "None" -AND RecipientTypeDetails -eq "UserMailbox"} | Enable-Mailbox -Archive
    ```

- После включения развертываемым архивации для вашей организации или для отдельного пользователя, архивный почтовый ящик преобразуется в архиве развертываемым, по достижении 90 ГБ архивного почтового ящика (в том числе папки элементов для восстановления). Может потребоваться до 30 дней дополнительные дискового пространства для подготовить к работе.
    
- После включения развертываемым архивации, оно не может быть отключена.
    
- Развертываемым архивации поддерживается для облачных архивных почтовых ящиков в гибридном развертывании Exchange для пользователей, имеющих локального основного почтового ящика. Тем не менее подключив развертываемым архивации для облачной архивного почтового ящика, вам не удается отдается этого архивного почтового ящика к локальной организации Exchange.
    
- Список клиентов Outlook, которые пользователи могут использовать для доступа к элементам в области дополнительное хранилище в почтовом ящике архива см в разделе «Требования к Outlook для доступа к элементам в архиве автоматически расширяется» [Общие сведения о неограниченное архивации в Office 365](unlimited-archiving.md#outlook-requirements-for-accessing-items-in-an-auto-expanded-archive) .
    
- Как объяснялось ранее, 10 ГБ добавляется в квоту хранилища основной архивного почтового ящика пользователя (и для папки элементов для восстановления Если почтовый ящик на хранение) при выполнении команды **Enable-Mailbox AutoExpandingArchive** . Это обеспечивает дополнительное хранилище до подготовки автоматически расширяется дискового пространства (который может занять до 30 дней). В этом дополнительного места хранения не добавляется, при запуске **AutoExpandingArchive Set-OrganizationConfig** для включения развертываемым архивации для всех почтовых ящиков в организации. Если включено автоматическое развертывание архивации для всей организации, но требуется для добавления дополнительных 10 ГБ дискового пространства для определенного пользователя, можно выполнить команду **Enable-Mailbox AutoExpandingArchive** на этот почтовый ящик. Обратите внимание, что появится сообщение об ошибке, о том, что развертываемым архивации уже была включена, но дополнительного места хранения добавляются к почтовому ящику.

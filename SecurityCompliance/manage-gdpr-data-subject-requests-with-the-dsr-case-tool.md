## <a name="optional-step-5-revise-the-built-in-search-query"></a><span data-ttu-id="6a3e8-273">(Необязательно) Шаг 5: Исправить встроенных поискового запроса</span><span class="sxs-lookup"><span data-stu-id="6a3e8-273">(Optional) Step 5: Revise the built-in search query</span></span>

При добавлении двух или нескольких уникальных условий для поискового запроса (условия, которые определяют различные свойства) оператор **AND** логически связаны эти условия.

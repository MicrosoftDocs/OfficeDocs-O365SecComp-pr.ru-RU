# <a name="detailed-properties-in-the-office-365-audit-log"></a>Подробные свойства в журнале аудита Office 365

При экспорте результаты поиска журнала аудита из безопасности Office 365 &amp; центре соответствия требованиям, у вас есть возможность загрузки всех результатов, которые соответствуют условиям поиска. Для этого выберите пункт **Экспорт результатов** \> **Загрузка всех результатов** на странице **поиска журнала аудита** в системы &amp; центре соответствия требованиям. Дополнительные сведения можно [Поиск в журнале аудита безопасности Office 365 &amp; центре соответствия требованиям](search-the-audit-log-in-security-and-compliance.md).
  
 Необработанные данные из журнала аудита объединенных копируется в запятой Office 365, все экспорта результатов для поиска журнала аудита, разделенных виде CSV-файл, это загружается на локальный компьютер. Этот файл содержит дополнительные сведения из записи журнала аудита в столбец с именем **подробностями**. Этот столбец содержит свойство Многозначный для нескольких свойств из записи журнала аудита. Каждой из пар **свойство: значение** в этом свойстве Многозначный, разделенных запятыми. 
  
В следующей таблице описываются свойства, которые включены, в зависимости от Office 365 службы, в которой происходит событие — в столбце **сведений о** многозначных свойствах. В столбце **службы Office 365, который имеет это свойство** указывается тип активности (пользователь или администратор), который включает в себя свойства и службы. Для получения дополнительных сведений об этих свойств или свойств, которые не могут быть доступны в этом разделе видеть [Схема управления активности API для Office 365](https://go.microsoft.com/fwlink/p/?LinkId=717993).

> Можно использовать Power запроса в Excel для разделения этот столбец в несколько столбцов, чтобы каждое свойство будет иметь свои собственные столбца. Это позволит программная сортировка и фильтрация по одному или нескольким из этих свойств. Чтобы узнать, как это сделать, обратитесь к разделу «Разделение столбец с разделителем» разбиению [столбец текста (Power запроса)](https://support.office.com/article/5282d425-6dd0-46ca-95bf-8e0da9539662).

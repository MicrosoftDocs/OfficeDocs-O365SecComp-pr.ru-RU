# <a name="create-activity-alerts-in-the-office-365-security-amp-compliance-center"></a>Создание оповещений активности безопасности Office 365 &amp; центре соответствия требованиям

Можно создать оповещение действие, которое будет отправлено уведомление по электронной почте при выполнении пользователем определенных действий в Office 365. Оповещения активности похожи на поиск события в журнале аудита Office 365, за исключением того, что будет отправлено сообщение электронной почты во время события для действия, который вы создали оповещение для происходит. 
  
 **Причины использования оповещений активности вместо поиск в журнале аудита?** Некоторые действия или операции, выполненные с определенным пользователям, которые действительно необходимо знать о возможно. Вместо того, чтобы не забудьте поиск в журнале аудита для этих действий, можно использовать действие оповещения для Office 365 отправьте сообщение электронной почты пользователям выполнять эти действия. Например можно создать оповещение активности для уведомления о том, когда пользователь удаляет файлы в SharePoint или создать оповещение для уведомления о том, когда пользователь окончательно удаляет сообщения из их почтовых ящиков. Уведомление по электронной почте, следует включены сведения о какие действие было выполнено и пользователя, выполнившего его. 
  
## <a name="before-you-begin"></a>Перед началом работы

- Необходимо назначить роль Конфигурация организации в системы &amp; центре соответствия требованиям для управления оповещениями активности. По умолчанию эта роль назначается группы ролей администратора соответствия требованиям и управления организацией. Дополнительные сведения о добавлении членов в группы ролей можно [предоставления пользователям доступа к Office 365 безопасность &amp; центре соответствия требованиям](grant-access-to-the-security-and-compliance-center.md).
    
- Вы (или другой администрирования) необходимо сначала включить ведение журнала аудита для вашей организации перед началом использования оповещений активности. Для этого просто щелкните **Начать запись действия администратора и пользователя** на странице **активности уведомления** . (Если не отображается ссылка, аудит уже был включен для вашей организации.) Также можно включить аудит на странице **поиска журнала аудита** в системы &amp; центре соответствия требованиям (последовательно выберите пункты **поиска &amp; расследования** \> **поиска журналов аудита**). Необходимо выполнить один раз для вашей организации.
    
- Страница **оповещений** в системы &amp; центре соответствия требованиям для создания оповещения только для операции, выполненные пользователями, которые перечислены в адресной книге вашей организации. На этой странице нельзя использовать для создания оповещений для операции, выполненные с внешними пользователями. 
    
- [Дополнительные сведения](#more-information) см раздел список распространенных сценариев (и определенные операции для отслеживания), что можно создать оповещения для. 
    
- Можно создать оповещения для те же действия, которые можно выполнить поиск в журнале аудита Office 365. 
    
## <a name="create-an-activity-alert"></a>Создание оповещения активности

1. Последовательно выберите пункты [https://protection.office.com](https://protection.office.com).
    
2. Войдите в Office 365 с помощью учетной записи рабочего или школы.
    
3. В левой области нажмите кнопку **оповещения**и нажмите кнопку **Управление оповещениями**.
    
4. На странице **активности уведомления** нажмите кнопку **добавить оповещение**.
    
    ![Добавление действия оповещения](media/53888bd5-9fa2-4398-8ccc-1a9dc72517ac.png)
  
5. Заполните следующие поля, чтобы создать оповещение.
    
    а. **имя** - введите имя для оповещения. Названия предупреждений должно быть уникальным в пределах организации.
    
    б. **Описание** (необязательно) — к оповещению, такие как действия и отслеживается, пользователи и пользователи, которые уведомлений по электронной почте отправляются. Описания позволяют быстро и легко с описанием назначения оповещения, чтобы другие "Администраторы".
    
    c. **это оповещение при отправки** - щелкните **это оповещение при отправки** , а затем настроить эти два поля:
    
    - **Действия** — выберите в раскрывающемся списке список для отображения действия, которые можно создать оповещение о. Это тот же список действий, который отображается, если поиск в журнале аудита Office 365. Вы можете выбрать одну или несколько конкретных операций или щелкнуть имя группы активности, чтобы выбрать все действия в группе. Описание этих действий обратитесь к разделу «Аудит действий» возможности [поиска в Office 365 безопасности & центре соответствия требованиям журнала аудита](search-the-audit-log-in-security-and-compliance.md#audited-activities). Когда пользователь выполняет никаких действий, которые добавлены в уведомлении, отправляется уведомление по электронной почте. 
    
     - **Пользователи** — установите этот флажок, а затем выберите одного или нескольких пользователей. Если пользователи в этом поле выполнить действия, которые вы добавили в список **действий** , оповещение будет отправлено. Заполняйте поле **Пользователи** для отправки оповещения в случае любого пользователя в организации выполняет операции, указанного идентификатором оповещение. 
    
    **Отправить это оповещение, чтобы** - d. нажмите кнопку **Отправить это оповещение**и щелкните в поле **Получатели** и введите имя для добавления пользователей, которые будут получать уведомления по электронной почте при выполнении пользователем (указанного в поле **Пользователи** ) активности (указано в Поле **действия** ). Обратите внимание на то, что они были добавлены в список получателей по умолчанию. Имя можно удалить из этого списка.
    
6. Нажмите кнопку **Сохранить** , чтобы создать оповещение. 
    
    Новое оповещение отображается в списке на странице **активности уведомления** . 
    
    ![На странице оповещений активности в безопасности отображается список предупреждений &amp; центре соответствия требованиям](media/02b774f2-1719-41de-bbc9-5e5b7576f335.png)
  
    Чтобы **на**установлено состояние оповещения. Обратите внимание, что также указываются получателей, которым будет получено уведомление по электронной почте, когда отправляется оповещение. 
  
## <a name="turn-off-an-activity-alert"></a>Отключить оповещение активности

Можно отключить оповещение активности, чтобы не отправляется уведомление по электронной почте. После отключения оповещение активности по-прежнему отображается в списке действие оповещения для вашей организации и по-прежнему можно просмотреть его свойства.
  
1. Последовательно выберите пункты [https://protection.office.com](https://protection.office.com).
    
2. Войдите в Office 365 с помощью учетной записи рабочего или школы.
    
3. В левой области нажмите кнопку **оповещения**и нажмите кнопку **Управление оповещениями активности**.
    
4. В списке оповещений для вашей организации щелкните оповещение, которое нужно отключить.
    
5. На странице " **Изменение оповещение** " выберите переключатель **на** , чтобы изменить состояние на **Off**и нажмите кнопку **Сохранить**.
    
    Чтобы **выключить**установлено состояние оповещения на страницах оповещения активности. 
    
Для включения активности оповещения, повторите эти действия и выберите переключатель **отключено** , чтобы изменить состояние на **на**.

При использовании этих командлетов для создания и изменения оповещений активности хранить в виду следующее:

  - Причины использовать командлеты для создания или редактирования оповещение активности — отправить уведомления по электронной почте пользователя не из вашей организации. В этом внешних пользователей будут отображены в список получателей оповещения. При удалении этого внешнего пользователя в оповещении этого пользователя не может быть восстановлена оповещение с помощью страницы **Изменить оповещение о** безопасности, но &amp; центре соответствия требованиям. Будет повторно добавить внешних пользователей, с помощью командлета **Set-ActivityAlert** или командлет **New-ActivityAlert** для добавления внешних пользователей же (или другой) для нового оповещения.

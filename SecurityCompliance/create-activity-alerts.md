Оповещения активности похожи на поиск события в журнале аудита Office 365, за исключением того, что будет отправлено сообщение электронной почты во время события для действия, который вы создали оповещение для происходит.</span><span class="sxs-lookup"><span data-stu-id="ff8e9-p101">You can create an activity alert that will send you an email notification when users perform specific activities in Office 365.  <span data-ttu-id="ff8e9-p102">**Причины использования оповещений активности вместо поиск в журнале аудита?** Некоторые действия или операции, выполненные с определенным пользователям, которые действительно необходимо знать о возможно. Вместо того, чтобы не забудьте поиск в журнале аудита для этих действий, можно использовать действие оповещения для Office 365 отправьте сообщение электронной почты пользователям выполнять эти действия. Например можно создать оповещение активности для уведомления о том, когда пользователь удаляет файлы в SharePoint или создать оповещение для уведомления о том, когда пользователь окончательно удаляет сообщения из их почтовых ящиков. Instead of having to remember to search the audit log for those activities, you can use activity alerts to have Office 365 send you an email message when users perform those activities. For example, you can create an activity alert to notify you when a user deletes files in SharePoint or you can create an alert to notify you when a user permanently deletes messages from their mailbox. The email notification sent to you includes information about which activity was performed and the user who performed it.</span></span>

# <a name="enable-mailbox-auditing-in-office-365"></a><span data-ttu-id="8f834-105">Включение аудита почтовых ящиков в Office 365</span><span class="sxs-lookup"><span data-stu-id="8f834-105">Enable mailbox auditing in Office 365</span></span>
  
<span data-ttu-id="8f834-p102">В Office 365 вы можете включить ведение журнала аудита почтовых ящиков, чтобы получать доступ к почтовым ящикам с помощью владельцев, делегатов и администраторов почтовых ящиков. По умолчанию аудит почтовых ящиков в Office 365 не включен. Это означает, что события аудита почтовых ящиков не будут отображаться в результатах поиска в журнале аудита Office 365 для действия почтовых ящиков. Но после включения ведения журнала аудита почтовых ящиков для почтового ящика пользователя можно выполнить поиск по журналу аудита для активности почтовых ящиков. Кроме того, если включено ведение журнала аудита почтовых ящиков, некоторые действия, выполняемые администраторами, представителями и владельцами, записываются по умолчанию. Чтобы выполнить вход в систему (а затем найти) дополнительные действия, обратитесь к этапу 3.</span><span class="sxs-lookup"><span data-stu-id="8f834-p102">In Office 365, you can turn on mailbox audit logging to log mailbox access by mailbox owners, delegates, and administrators. By default, mailbox auditing in Office 365 isn't turned on. That means mailbox auditing events won't appear in the results when you search the Office 365 audit log for mailbox activity. But after you turn on mailbox audit logging for a user mailbox, you can search the audit log for mailbox activity. Additionally, when mailbox audit logging is turned on, some actions performed by administrators, delegates, and owners are logged by default. To log (and then search for) additional actions, see Step 3.</span></span>

## <a name="step-2-enable-mailbox-audit-logging"></a><span data-ttu-id="8f834-124">Этап 2. Включение журнала аудита почтовых ящиков</span><span class="sxs-lookup"><span data-stu-id="8f834-124">Step 2: Enable mailbox audit logging</span></span>

<span data-ttu-id="8f834-p104">После подключения к организации Exchange Online используйте PowerShell, чтобы включить ведение журнала аудита почтовых ящиков для почтового ящика. <span data-ttu-id="8f834-128">В этом примере мы включили ведение журнала аудита для почтовых ящиков всех пользователей в организации.</span><span class="sxs-lookup"><span data-stu-id="8f834-128">This example enables mailbox audit logging for all user mailboxes in your organization.</span></span>

# <a name="enable-mailbox-auditing-in-office-365"></a><span data-ttu-id="f3503-105">Включение аудита почтовых ящиков в Office 365</span><span class="sxs-lookup"><span data-stu-id="f3503-105">Enable mailbox auditing in Office 365</span></span>
  
<span data-ttu-id="f3503-p102">В Office 365 можно включить ведение журнала аудита почтовых ящиков для регистрировать доступ владельцев, делегатов и администраторов. По умолчанию аудита почтовых ящиков в Office 365 не включено. Это означает, что при выполнении поиска журнал аудита Office 365 для почтового ящика действия аудита почтовых ящиков не будет отображаться в результатах. Но после включения почтового ящика ведения журнала аудита для почтового ящика пользователя можно выполнить поиск журнала аудита для почтового ящика активности. Кроме того когда аудита почтовых ящиков ведения журнала, некоторые действия, выполняемые администраторами, делегаты, и владельцев заносятся по умолчанию. Для входа (и затем найдите) Дополнительные действия, содержатся в шаге 3.</span><span class="sxs-lookup"><span data-stu-id="f3503-p102">In Office 365, you can turn on mailbox audit logging to log mailbox access by mailbox owners, delegates, and administrators. By default, mailbox auditing in Office 365 isn't turned on. That means mailbox auditing events won't appear in the results when you search the Office 365 audit log for mailbox activity. But after you turn on mailbox audit logging for a user mailbox, you can search the audit log for mailbox activity. Additionally, when mailbox audit logging is turned on, some actions performed by administrators, delegates, and owners are logged by default. To log (and then search for) additional actions, see Step 3.</span></span>

## <a name="step-2-enable-mailbox-audit-logging"></a><span data-ttu-id="f3503-124">Этап 2. Включение журнала аудита почтовых ящиков</span><span class="sxs-lookup"><span data-stu-id="f3503-124">Step 2: Enable mailbox audit logging</span></span>

<span data-ttu-id="f3503-p104">После подключения к организации Exchange Online с помощью PowerShell для включения почтового ящика ведения журнала аудита для почтового ящика. <span data-ttu-id="f3503-127">В этом примере включается почтового ящика ведения журнала аудита для почтового ящика пользователя Марты Артемьевой.</span><span class="sxs-lookup"><span data-stu-id="f3503-127">This example enables mailbox audit logging for Pilar Pinilla's mailbox.</span></span>

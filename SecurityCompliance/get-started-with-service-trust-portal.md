# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Начало работы с Microsoft Service Trust Portal

Портал управления безопасностью службы Microsoft, предоставляет выбор содержимого, средств и другие ресурсы, посвященные безопасности, конфиденциальности и соответствия рекомендациям Майкрософт.
  
## <a name="accessing-the-service-trust-portal"></a>Доступ к портале управления безопасностью

Портал управления безопасностью службы содержит сведения о реализации корпорации Майкрософт элементов управления и процессы, которые указанные защитить наших облачных служб и данных клиента. Для доступа к некоторые ресурсы на портале управления безопасностью, необходимо выполните вход в качестве пользователя с разрешением с помощью учетной записи Microsoft облачных служб (учетную запись организации Azure Active Directory или учетной записи Майкрософт) и просмотрите и примите корпорации Майкрософт Соглашения о неразглашении для соответствия требованиям материалов.
  
**Существующие клиенты могут доступа к порталу доверия службы** в [https://aka.ms/STP](https://aka.ms/STP) с одним из следующих online подписок (пробной версии или платная): 
  
- Office 365

- Dynamics 365

- Azure
    
 > [!NOTE]
 > Azure Active Directory учетных записей, связанных с организациями имеют доступ для всех документов и функции как Compliance Manager. Учетные записи Майкрософт, созданные для личного использования ограничен доступ к содержимому на портале управления безопасностью. 
  
**Новых клиентов и клиентов, оценка Microsoft online services**
  
Чтобы создать новую учетную запись или создайте пробной учетной записи, используйте одну из следующих форм регистрации (также используется для пробного учетных записей) для получения доступа к STP.
  
- Регистрация для новых [пробной учетной записи Office 365 для бизнеса](https://go.microsoft.com/fwlink/p/?LinkID=507653) или новый [пробной учетной записи Office 365 для предприятия](https://go.microsoft.com/fwlink/p/?LinkID=698279)

- Зарегистрируйтесь для нового [Dynamics 365 пробной учетной записи](https://go.microsoft.com/fwlink/?LinkId=252780)

- Зарегистрируйтесь для нового [Azure пробной учетной записи](https://go.microsoft.com/fwlink/?LinkId=722737). 
    
При подписке на бесплатную пробную версию или подписки необходимо включить Azure Active Directory для поддержки доступа к STP.
  
## <a name="using-the-service-trust-portal"></a>Использование портала управления безопасностью службы

Компоненты портала доверия службы и содержимое доступны из главного меню.
  
![Меню портале управления безопасностью](media/STPMenus1.png)

В следующих разделах каждый элемент в главное меню.
  
### <a name="service-trust-portal"></a>Service Trust Portal

Ссылка на **Портале управления безопасностью** будет отображаться на домашнюю страницу. Оно предоставляет возможность быстрого вернуться на домашнюю страницу.

### <a name="compliance"></a>Соответствие требованиям

Предоставляет следующие связанные с соответствием средства и ресурсы:

- Отображается **отчетов аудита** - список независимых отчетов аудита и оценки на облачные службы корпорации Майкрософт. Эти отчеты предоставляют сведения об облаке Майкрософт служб соответствия стандартам защиты данных и нормативным требованиям, такие как:
  
    - Международной организации по стандартизации (ISO)
    - Элементы управления организации службы (SOC)
    - Национальный института стандартов и технологий (NIST)
    - Федеральное риск и программе управления авторизации (FedRAMP)
    - Нормы защиты данных General (GDPR)

  Сведения об отчетах аудита и ссылка на странице **отчетов аудита** также отображается на домашней странице портала управления безопасностью служб. 
  
- **Compliance Manager** - Compliance Manager — это средство оценки рисков на основе рабочего процесса, который позволяет отслеживать, назначение и проверка соответствия нормативным требованиям вашей организации действия связанные с облачные службы Майкрософт, такие как Office 365 Dynamics 365 и Azure. Используйте диспетчер соответствия требованиям для управления соответствия нормативным требованиям в рамках модели общей ответственностью облака. Дополнительные сведения можно [Использовать диспетчер соответствия для соответствия защита данных и нормативным требованиям, используя облачные службы Майкрософт](meet-data-protection-and-regulatory-reqs-using-microsoft-cloud.md).

Нажмите кнопку с изображением увеличительного стекла в правом верхнем углу страницы портала управления безопасностью служб, чтобы развернуть поле, введите критерии поиска и нажмите клавишу **Ввод**.

![Service Trust Portal: поле для ввода поисковых запросов](media/7c5cd817-3d62-420b-adb4-76e33fef941f.png)
  
 Отображается страница **поиска** , с помощью условия поиска, отображаемых в поле «Поиск» и результаты поиска из списка ниже.
  
![Service Trust Portal: поиск документов с примененным фильтром](media/86b754e1-c63c-4514-89ac-d014bf334140.png)

По умолчанию поиска возвращает результаты документа. Результаты можно с помощью списков раскрывающийся список, чтобы настроить список документов, которые отображаются. В результатах поиска отображаются дата оценки был создан, имя оценки группировки, соответствующую службу Microsoft Cloud и является ли элемент управления Microsoft или управляемого клиента. Щелкните имя элемента управления для просмотра элемента управления в оценки в диспетчере соответствия требованиям.

## <a name="my-library"></a>Библиотеки

Используйте функцию библиотеку для добавления на страницу библиотеку документов и ресурсы на портале управления безопасностью. Это позволяет получить доступ к документам, применяемых в одном месте.  Для добавления документа в библиотеку личные, выберите в меню **...** в правой части документа и выберите команду **Добавить в библиотеку**. Можно добавить несколько документов в библиотеку личных, установив флажок рядом с одним или несколькими документами и затем нажмите кнопку **Сохранить в библиотеке** в верхней части страницы.

Кроме того функцию уведомлений позволяет настроить личные библиотеки, чтобы сообщение электронной почты отправляется всякий раз, когда документ, который вы добавили в библиотеку личных обновлений Microsoft. Чтобы настроить уведомления, перейдите к библиотеке личных и нажмите кнопку **Параметры уведомлений**. Можно выбрать частоту уведомлений и указать адрес электронной почты в вашей организации, чтобы отправлять уведомления. Уведомления по электронной почте включают ссылки на документы, которые были обновлены и краткое описание обновления.

Обратите внимание, что мы идентификации документы в библиотеке личных, которые были обновлены в рамках за последние 30 дней, независимо от того, является ли включить уведомления. Краткое описание обновления также отображается в окне подсказки. 

## <a name="starter-packs"></a>Пакеты Starter

Пакеты Starter представляют собой комплект Microsoft curated документация по облачные службы Майкрософт для конкретных отраслей. На данный момент портала управления безопасностью служб предлагает следующие три starter пакетов для организаций, финансовые услуги. Эти пакеты starter помочь организациям оценка и оценки безопасности, соответствия требованиям и конфиденциальности в облаке Майкрософт и позволяют реализовать облачные службы Майкрософт в очень регулируемого финансовых служб.

- **Пакет Starter оценки** — используется для первых ознакомительная версия Microsoft cloud для организаций, финансовые услуги.

- **Пакет начального оценки** - после оценки, используйте контрольные списки и другую информацию в этот пакет starter организации оценки рисков, относящихся к безопасности, соответствия требованиям и конфиденциальности.

- **Пакет начального аудита** - пользователя этот пакет starter для руководство по использованию аудита элементов управления и других средств для помощи в реализации облачные службы Майкрософт, чтобы помогает снижения риска риска вашей организации.

Для доступа к эти пакеты starter, перейдите на **портале управления безопасностью > отраслях & областей > > решений для отрасли финансовые услуги**. Можно открыть или документов загрузки из начального пакетом обновления или сохранять их на мой библиотеки.

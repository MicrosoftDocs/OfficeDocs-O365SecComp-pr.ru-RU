# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Начало работы с порталом Microsoft доверия службы

Microsoft службы управления безопасностью портала (STP), предоставляет выбор содержимого, средств и другие ресурсы, посвященные безопасности, конфиденциальности и соответствия рекомендациям Майкрософт. Включает отчеты аудита независимой сторонних производителей online services корпорации Майкрософт и сведения о как наших интерактивных служб может помочь вашей организации обслуживания и отслеживать соответствие требованиям стандартов, законам и положения, таких как:
  
- Международной организации по стандартизации (ISO)
    
- Элементы управления организации службы (SOC)
    
- Национальный института стандартов и технологий (NIST)
    
- Федеральное риск и программе управления авторизации (FedRAMP)
    
- Нормы защиты данных General (GDPR)
  
## <a name="accessing-the-service-trust-portal"></a>Доступ к портале управления безопасностью

STP содержит подробные сведения о реализации корпорации Майкрософт элементов управления и процессы, которые указанные защитить наших облачных служб и данных клиента. Для доступа к некоторых STP материалы, необходимо выполните вход в качестве пользователя с разрешением с помощью учетной записи Microsoft облачных служб (учетную запись организации Azure Active Directory или учетной записи Майкрософт) и просмотрите и примите соглашение о неразглашении Microsoft для Соответствие требованиям материалы.
  
 **Существующие клиенты могут получить доступ к STP** в [https://aka.ms/STP](https://aka.ms/STP) с одним из следующих online подписок (пробной версии или платная): 
  
- **Office 365**
    
- **Dynamics 365**
    
- **Azure**
    
> [!NOTE]
> Azure Active Directory учетных записей, связанных с организациями имеют доступ для всех документов и функции как Compliance Manager. Учетные записи Майкрософт, созданные для личного использования ограничен доступ к содержимому на портале управления безопасностью. 
  
 **Новых клиентов и клиентов, оценка Microsoft online services**
  
Чтобы создать новую учетную запись или создайте пробной учетной записи, используйте одну из следующих форм регистрации (также используется для пробного учетных записей) для получения доступа к STP.
  
- Регистрация для новых [пробной учетной записи Office 365 для бизнеса](https://go.microsoft.com/fwlink/p/?LinkID=507653) или новый [пробной учетной записи Office 365 для предприятия](https://go.microsoft.com/fwlink/p/?LinkID=698279)
    
- Зарегистрируйтесь для нового [Dynamics 365 пробной учетной записи](https://go.microsoft.com/fwlink/?LinkId=252780)
    
- Зарегистрируйтесь для нового [Azure пробной учетной записи](https://go.microsoft.com/fwlink/?LinkId=722737). 
    
При подписке на бесплатную пробную версию или подписки необходимо включить Azure Active Directory для поддержки доступа к STP.

### <a name="trust-documents"></a>Документы о защите данных

Обеспечение безопасной портале доверие обеспечивает доступ к богатым реализации безопасности и информации о структуре с целью облегчения соответствовать нормативным требованиям, общие сведения о как облачные службы Майкрософт хранение данных. Чтобы просмотреть содержимое, выберите параметр в меню.
  
 - **Отчеты аудита** содержит независимые аудита, а также оценки отчетов по Microsoft облачных служб соответствия стандартам защиты данных и нормативным требованиям, в том числе: 
  
 - **Защита данных** содержит доверять документы для загрузки, сведения о том, как работает Microsoft Azure, Dynamics 365 и Office 365. 
  
 - **Azure безопасность и соответствие требованиям чертежей** предлагает параметр Включить ключ безопасности и соответствия требованиям решения и поддержки, предприятий вертикали отрасли, которые ускорить загрузку для клиентов с данными регулируемого или ограниченных и внедрения облака. 
  
### <a name="regional-compliance"></a>Соответствие региональным требованиям

 - **Чешская Республика** предоставляет юридических опытом на соответствие законодательство Чешская Республика веб-службами Microsoft. 
  
 - **Польша** предоставляет юридических опытом на соответствие требованиям веб-службами Майкрософт с законами Польши. 
  
 - **Румыния** предоставляет юридических опытом на соответствие требованиям веб-службами Майкрософт с законами Румыния. 
  
 - **Испания** предоставляет юридических опытом на соответствие требованиям веб-службами Майкрософт с законами Испания.

### <a name="search"></a>Сервер поиска

![Портал управления безопасностью службы - поля поиска](media/7c5cd817-3d62-420b-adb4-76e33fef941f.png)
  
Нажмите кнопку увеличительного стекла в правом верхнем углу страницы, чтобы развернуть поля ввода поиска, введите критерии поиска и нажмите клавишу ВВОД. Элемент управления поиска будут отображаться с условие поиска в области ввода поле поиска, и под будут отображаться результаты поиска.
  
По умолчанию поиска возвращает результаты документов, и можно использовать раскрывающиеся списки фильтра, чтобы настроить список документов, которые отображаются, чтобы добавить или удалить результаты поиска из представления. Можно использовать несколько атрибутов фильтра в то же время, чтобы уменьшить количество возвращаемых документов для конкретного облачные службы, категории соответствия или по обеспечению безопасности, регионах мира или отраслях. Щелкните ссылку для загрузки документа с именем документа.
  
![Портал управления безопасностью службы - поиска документов с применением фильтра](media/86b754e1-c63c-4514-89ac-d014bf334140.png)
  
Щелкните ссылку Compliance Manager для отображения результатов поиска для элементов управления оценки Compliance Manager. Результаты поиска перечисленных отображения даты оценки был создан, имя оценки группировки, которые применяются облачная служба и элементы управления, являются ли Microsoft или управляемого клиента.
  
![Портал управления безопасностью службы - поиска на элементах управления Compliance Manager](media/bafb811a-68ce-40b5-ad16-058498fe5439.png)
  
> [!NOTE]
> Отчеты о портале управления безопасностью и документы доступны для загрузки по крайней мере двенадцать месяцев после публикации или до новой версии документа становится доступной.

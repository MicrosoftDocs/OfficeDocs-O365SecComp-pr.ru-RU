# <a name="search-for-ediscovery-activities-in-the-office-365-audit-log"></a>Поиск мероприятий обнаружения электронных данных в журнале аудита Office 365

Поиск и действий, связанных с обнаружения электронных данных, которые выполняются в Office 365 безопасность содержимого &amp; центре соответствия требованиям или запустив соответствующие Windows PowerShell командлеты записываются в журнал аудита Office 365. Событий при Администраторы или Администраторы соответствия требованиям (или любой пользователь, назначенный eDiscovery разрешения) выполните следующие поиска контента и задачи, связанные с eDiscovery безопасности Office 365 &amp; центре соответствия требованиям:
  
- Создание и управление вариантах eDiscovery
    
- Создание, запуск и изменение контента поиска
    
- Для выполнения действий поиска контента, например Просмотр, экспорт и удаление результатов
    
- Настройка разрешений фильтрации для поиска контента
    
- Управление роль администратора обнаружения электронных данных
    
> [!IMPORTANT]
> Действий, описанных в этой статье приведены только в результате eDiscovery задачи, выполняемые с помощью безопасности &amp; центре соответствия требованиям. обнаружение электронных данных задачи, которые были созданы с помощью средства обнаружения электронных данных на месте в Exchange Online или центр обнаружения электронных данных в SharePoint Online не включаются. 
  
Дополнительные сведения о поиске в журнал аудита Office 365 разрешения, необходимые и экспорт результатов поиска, просмотрите [Поиск в журнале аудита безопасности Office 365 &amp; центре соответствия требованиям](search-the-audit-log-in-security-and-compliance.md).
  
## <a name="how-to-search-for-and-view-ediscovery-activities"></a>Поиск и Просмотр мероприятий обнаружения электронных данных

В настоящее время необходимо выполнить некоторые действия определенного для просмотра действий обнаружения электронных данных в журнале аудита Office 365. Вот как.
  
1. Перейдите по ссылке [https://protection.office.com](https://protection.office.com).
    
2. Войдите в Office 365 с помощью своей рабочей или учебной учетной записи.
    
3. В левой области щелкните **поиска &amp; расследования**и нажмите кнопку **Поиск журнала аудита**.
    
4. В раскрывающемся списке **действий** в разделе **действия обнаружения электронных данных**, щелкните одну или несколько операций для поиска. Или нажать кнопку **действия eDiscovery** для поиска всех действий, связанных с обнаружения электронных данных. 
    
    > [!NOTE]
    > Список раскрывающийся список действий также включает группу с именем **действия командлета обнаружения электронных данных** , возвращаемых записей из журнала аудита командлет операций. 
  
5.  Выберите дату и время диапазона для отображения обнаружения электронных данных события, произошедшие в течение этого периода. 
    
6. В поле **Пользователи** выберите одного или нескольких пользователей для отображения результатов поиска. Оставьте это поле пустым для возвращения записей для всех пользователей. 
    
7. Нажмите кнопку **поиска** , чтобы запустить поиск, используя условия поиска. 
    
8. После отображения результатов поиска, щелкните **результатов фильтра** , чтобы отфильтровать или отсортировать результирующий активности. К сожалению явно исключить некоторые действия нельзя использовать фильтрацию. 
    
9. Чтобы просмотреть сведения об активности щелкните запись действия в списке результатов поиска. 
    
    Во время работы **сведений** страницы отображается, содержащий подробные свойства из записи журнала событий. Чтобы открыть Дополнительные сведения, нажмите кнопку **Дополнительные сведения**. Описание этих свойств обратитесь к разделу [Подробные свойства для операций обнаружения электронных данных](#detailed-properties-for-ediscovery-activities) .

# <a name="supported-file-types-in-advanced-ediscovery"></a>Поддерживаемые типы файлов в Advanced eDiscovery

Расширенное обнаружение электронных данных поддерживает множество типов файлов с множеством различных уровней, описанных в следующей таблице. Этот список не завершен, и мы добавим новые типы файлов, когда мы продолжим проверочное тестирование. Таблицы указывают, поддерживает ли тип файлов для извлечения текста (OCR для изображений), которые можно просмотреть в собственном средстве просмотра и поддерживаются в средстве просмотра меток в Advanced eDiscovery.

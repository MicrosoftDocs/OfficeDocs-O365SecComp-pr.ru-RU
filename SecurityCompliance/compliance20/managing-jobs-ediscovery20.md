# <a name="manage-jobs-in-advanced-ediscovery-preview"></a>Управление заданиями в Advanced eDiscovery (Предварительная версия)

Ниже приведен список заданий, отслеживаемых на вкладке " **задания** " в случае Advanced EDiscovery (Предварительная версия).

| Тип задания | Описание | Рабочие нагрузки, к которым применяется это задание |
| :- | :- | :- |
| Повторное индексирование данных хранитель | Когда хранитель добавляется к случаю, все частично индексированные элементы переиндексируются с помощью расширенного индексирования обнаружения электронных данных, дополнительные сведения см в разделе [Расширенная индексация данных хранитель](indexing-custodian-data.md). | Advanced eDiscovery |
| Оценка результатов поиска | Оценка результатов поиска это результат создания и выполнения поиска для рабочих нагрузок, таких как Exchange, SharePoint и OneDrive для бизнеса.  Оценки включают в себя статистику, например количество элементов, возвращенных при поиске и размере.  Дополнительные сведения о поиске можно найти [в статье сбор данных для обращения в Advanced eDiscovery](collecting-data-for-ediscovery.md). | Advanced eDiscovery |
| Подготовка предварительного просмотра поиска | Подготовка заданий предварительного просмотра поиска это результат создания и выполнения поиска для рабочих нагрузок, таких как Exchange, SharePoint и OneDrive для бизнеса.  Доступен предварительный просмотр результатов поиска, позволяющий определить еффикаци поиска.  Дополнительные сведения о поиске можно найти [в статье сбор данных для обращения в Advanced eDiscovery](collecting-data-for-ediscovery.md). | Advanced eDiscovery |
| Добавление данных в рабочий набор | Добавление данных к рабочим заданиям происходит при добавлении результатов поиска в рабочий набор.  Дополнительные сведения об управлении рабочими множествами [в разделе Manage working sets in Advanced eDiscovery](managing-working-sets.md). | Advanced eDiscovery |
| Добавление данных к другому рабочему набору | Добавление данных в другие задания рабочего набора создается, когда пользователь выбирает Добавление данных из одного рабочего набора в другой. | Advanced eDiscovery |
| Добавление данных, отличных от Office 365, в рабочий набор | Добавление данных, отличных от Office 365, в задания рабочего набора включает обработку и добавление данных в рабочий набор, отправленный с клиентского компьютера, который может быть релевантен для случая, для получения дополнительных сведений о добавлении данных, отЛичных от Office 365, в раздел Загрузка данных, отличных [от office 365, в Рабочий набор](load-non-office365-data.md). | Advanced eDiscovery |
| Подготовка к устранению ошибок |  | Advanced eDiscovery |
| Добавление исправленных данных в рабочий набор | Добавление исправленных данных в рабочий набор ссылается на задания, созданные при исправлении ошибочных файлов, которые загружаются обратно в рабочий набор Office 365 Advanced eDiscovery.  Для получения дополнительных сведений об исправлении ошибок в разделе [устраненИе ошибок при обработке данных](error-remediation.md). | Advanced eDiscovery |
| Сравнение наборов нагрузки | Сравнение наборов нагрузок позволяет пользователям проверить различия между загрузкой в рабочие наборы, чтобы получить дополнительные сведения о наборах нагрузки: [Manage Load setss](manage-load-sets.md). | Advanced eDiscovery |
| Маркировка документов | РасСтановка тегов в документах задания выполняются, когда несколько документов в рабочем наборе размечены тегами.  Для получения дополнительных сведений ознакомьтесь [с разметкой документы в рабочем наборе](tagging-documents.md). | Advanced eDiscovery |
| Выполнение аналитики | Выполнение заданий аналитики относится к задачам обработки, в том числе по дублированию идентификации, анализу и темам, анализу и темам, для получения дополнительной информации о аналитике: [анализ данных в рабочем наборе в Advanced eDiscovery](analyzing-data-in-working-set.md). | Advanced eDiscovery |
| Подготовка данных для экспорта | Подготовка данных для заданий экспорта — это результат экспорта данных из рабочего набора, дополнительные сведения о экспорте приведены [в статье Export Data Case in Advanced eDiscovery](exporting-data-ediscover20.md). | Advanced eDiscovery |

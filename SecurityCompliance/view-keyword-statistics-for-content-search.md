# <a name="view-keyword-statistics-for-content-search-results"></a>Просмотр статистики ключевых слов для результатов поиска контента

После создания и запуска поиска контента можно просмотреть статистику по предполагаемым результатам поиска. Сюда входит сводка результатов поиска (аналогично обзору оценочных результатов поиска в области сведений), статистики запросов, таких как количество расположений контента с элементами, которые отвечают поисковому запросу, а также имена расположений содержимого. с наиболее подходящими элементами. Можно отобразить статистику для одного или нескольких поисков контента. Это позволяет быстро сравнивать результаты для нескольких поисков и принимать решения относительно эффективности поисковых запросов.
  
Кроме того, вы можете настроить новые и существующие операции поиска, чтобы получить статистику для каждого ключевого слова в поисковом запросе. Это позволяет сравнить количество результатов для каждого ключевого слова в запросе и сравнить статистику ключевых слов из нескольких поисков.
  
Вы также можете скачать статистику поиска и статистику ключевых слов в CSV-файл. Это позволяет использовать функции фильтрации и сортировки в Excel для сравнения результатов и подготовки отчетов для результатов поиска.
  
## <a name="get-statistics-for-content-searches"></a>Получение статистики для поисков контента

Чтобы отобразить статистику для поисков контента:
  
1. В центре безопасности _амп_ соответствия требованиям выберите **** \> поиск **контента**.
    
2. В списке поиска выберите один или несколько поисков и нажмите кнопку Поиск **статистики**![поиска](media/9bf56d43-25bf-4f53-a4be-f4d55102310c.png).
    
    ![Выберите несколько поисков и нажмите кнопку Статистика поиска.](media/1195c6c3-2e00-469d-8c29-85c1c7ebe6c7.png)
  
3. На странице **Статистика поиска** щелкните одну из следующих ссылок, чтобы отобразить статистику выбранных поисков. 
    
    **Summary**
    
    На этой странице отображаются статистические данные, аналогичные тем, которые отображаются в области сведений на странице " **Поиск контента** ". Отображается статистика по всем выбранным поисковым запросам. Обратите внимание, что вы также можете повторно выполнить выбранные операции поиска на этой странице, чтобы обновить статистику. 
    
    ![Сводка статистики для выбранных поисков](media/abb663eb-b3d6-4f4c-a99f-55d20b0848af.png)
  
    a.  Имя поиска контента. Как было сказано ранее, можно отобразить и сравнить статистику для нескольких поисков.
    
    b. Тип расположения контента, в котором выполнялся поиск. В каждой строке отображается статистика для почтовых ящиков, сайтов и общедоступных папок из указанного поиска.
    
    В. Количество расположений контента, содержащих элементы, которые совпадают с поисковым запросом. Для почтовых ящиков Эта статистика также включает количество архивных почтовых ящиков, которые содержат элементы, которые совпадают с поисковым запросом.
    
    Г. Общее количество элементов всех указанных расположений контента, которые совпадают с поисковым запросом. Примеры типов элементов: сообщения электронной почты, элементы календаря и документы. Если элемент содержит несколько экземпляров ключевых слов, для которых выполняется поиск, оно подсчитывается только один раз в общее число элементов. Например, если вы ищете слова "Stock" или "мошенничество", а сообщение электронной почты содержит три экземпляра слова "Stock", оно подсчитывается только один раз в столбце **Items (элементы** ). 
    
    Д. Общий размер всех элементов, найденных в указанном расположении содержимого, которые совпадают с поисковым запросом. 
    
    **Queries**
    
    На этой странице отображается статистика поискового запроса.
    
    ![Статистика запросов поиска для выбранных поисков](media/dc817526-dfb9-43d3-a14c-4c58077eb7bb.png)
  
    a. Имя поиска контента, для которого строка содержит статистику запросов.
    
    b. Тип расположения контента, к которому относится статистика запроса.
    
    В. В этом столбце указывается, к какой части поискового запроса относятся статистические данные. **Основной** указывает весь поисковый запрос. Если вы используете список ключевых слов при создании или изменении поискового запроса, в эту таблицу включены статистические данные для каждого компонента запроса. Для получения дополнительных сведений ознакомьтесь с разделом [Получение статистики ключевых слов для поиска контента](#get-keyword-statistics-for-content-searches) этой статьи. 
    
    Г. В этом столбце содержится фактический запрос поиска, выполняемый средством поиска контента. Обратите внимание, что средство автоматически добавляет несколько дополнительных компонентов в создаваемый запрос. 

    - При поиске всего содержимого в почтовых ящиках (без указания ключевых слов) фактическим ключевым запросом Word `size>=0` считается то, что возвращаются все элементы. 
    
     - При поиске на сайтах SharePoint Online и OneDrive для бизнеса добавляются два следующих компонента:
    
          **Не IsExternalContent: 1** — исключение любого контента из локальной организации SharePoint. 
    
          **Not исоненотепаже: 1** -исключает все файлы OneNote, так как они будут дублироваться любого документа, который соответствует поисковому запросу. 

    
    Д. Число расположений содержимого (указывается в столбце * * Location Type * * * * * * * * * *), содержащие элементы, которые совпадают с поисковым запросом, указанным в столбце **запрос** . 
    
    Е. Количество элементов (из указанного расположения контента), которые совпадают с поисковым запросом, указанным в столбце " **запрос** ". Как описывалось ранее, если элемент содержит несколько экземпляров ключевого слова, в котором выполняется поиск, он подсчитывается только один раз в этом столбце. 
    
    финансов. Общий размер всех найденных элементов (в указанном расположении содержимого), которые отвечают поисковому запросу в столбце **запроса** . 
    
    **Основные расположения**
    
    На этой странице отображается статистика по количеству элементов, которые совпадают с поисковым запросом в каждом расположении контента, в котором выполнялся поиск. Отобразятся первые 1 000 расположений. Если вы просматриваете статистику для нескольких поисков, отображаются первые 1 000 расположений для каждого поиска. Обратите внимание, что расположение контента не включается на эту страницу, если оно не содержит элементов, которые совпадают с поисковым запросом.
    
    ![Статистика по количеству элементов, найденных в расположениях контента, в которых выполнялся поиск](media/35a820b0-85d9-45d1-9a0c-c74bec803e67.png)
  
    a. Имя расположения контента.
    
    b. Тип расположения контента, к которому относится статистика расположения.
    
    В. Для каждого поиска, для которого отображается статистика, существуют столбцы. В этом столбце отображается число (и общий размер) элементов, которые совпадают с поисковым запросом в каждом расположении контента. Обратите внимание, что когда вы отображаете статистику для нескольких поисков, в этом столбце "НД" указывает, что расположение контента не было включено в этот поиск. 

## <a name="get-keyword-statistics-for-content-searches"></a>Получение статистики ключевых слов для поиска контента

Как описано выше, на странице " **запросы** " отображается запрос поиска и число (и размер) элементов, которые совпадают с запросом. Если при создании или редактировании поискового запроса используется список ключевых слов, можно получить расширенную статистику, которая показывает, сколько элементов совпадают с ключевыми словами или ключевыми словами. Это поможет быстро определить, какие части запроса являются самыми эффективными (и, как минимум). Например, если ключевое слово возвращает большое количество элементов, можно выбрать уточнение запроса ключевых слов для сужения результатов поиска. Вы можете настроить список ключевых слов при создании или изменении поиска контента. 


Создание списка ключевых слов и просмотр статистики ключевых слов для поиска контента:
  
1. В центре безопасности _амп_ соответствия требованиям выберите **** \> поиск **контента**.
    
2. В списке Поиск контента щелкните и поиск, а затем щелкните **изменить** ![значок](media/ebd260e4-3556-4fb0-b0bb-cc489773042c.gif)редактирования.
    
3. Нажмите кнопку **запрос** , а затем выполните следующие действия: 
    
    ![Щелкните флажок Показать список ключевых слов и введите ключевое слово в каждой строке](media/73ef46dd-3d5c-415d-b5e7-c3559caaafe2.png)
  
    a. Установите флажок **Показать список ключевых слов** . 
    
    b. Введите ключевое слово или этап ключевого слова в строку в таблице ключевые слова. Например, введите **бюджет** в первую строку, а затем введите **Security** во второй строке. 
    
4. После добавления ключевых слов для поиска и получения статистики нажмите кнопку **Поиск** , чтобы запустить измененный Поиск. 
    
5. По завершении поиска выберите его в списке поиска и нажмите кнопку **** ![Поиск статистики поиска](media/9bf56d43-25bf-4f53-a4be-f4d55102310c.png). Кроме того, можно отображать и сравнивать статистику ключевых слов для нескольких поисков.
    
6. На странице **Статистика поиска** нажмите кнопку **запрос** , чтобы отобразить статистику ключевых слов для выбранных поисков. 
    
    ![Отображаются статистические данные для каждого ключевого слова для выбранных поисков](media/e7910fa9-af93-4df9-92d0-e1e3e089e14f.png)
  
    Как показано на предыдущем снимке экрана, отображаются статистические данные по каждому ключевому слову; в том числе: 
    
    - Статистика по ключевым словам для каждого типа расположения контента, включенного в поиск.
    
    - Фактический поисковый запрос для каждого ключевого слова, включающий все условия из поискового запроса. 
    
    - Полный поисковый запрос (определенный как **основной** в столбце " **часть** ") и статистика для полного запроса. Обратите внимание на то, что статистика отображается на странице **сводки** . 

> [!NOTE]
> Для сокращения проблем, вызванных списками с большими ключевыми словами, вы можете ограничить до 20 строк в списке ключевых слов поискового запроса.
